Project Description

Overview

Our project centered on the investigation of cross-lingual embedding alignment using anchor points, even in the absence of proper dictionaries. The primary goal was to enable the training of a language model for a specific downstream task in one language and its effective application in another.

Key details

  • Collaborative effort with a colleague, where I created anchor points and found datasets for downstream evaluation.
  • Pursued two distinct approaches for generating anchor points: Utilized aligned corpora from MultiParaCrawl to extract named entities in both languages. Identified an equal number of named entities, treated them as translations, and employed them as anchor points. Additionally, harnessed BabelNet, a semantic network linking concepts and named entities across multiple languages, to expand the pool of anchor points.

Project Outcome

We tested various anchor points in a downstream task, leading to the following findings and insights:

  • Hand-checked dictionaries yielded the best results, highlighting their importance.
  • In the absence of dictionaries, alternative anchor points delivered satisfactory results, albeit slightly less effective.
  • BabelNet-derived anchor points outperformed those from named entities, showing superior performance.
  • Highest effectiveness observed when aligning similar languages, leveraging their linguistic similarities.
