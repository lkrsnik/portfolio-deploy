Project Description

In the Cordex project, we developed a Python library dedicated to extracting collocations from text in diverse formats. Utilizing multicore processing and pypy, the library efficiently identifies collocations based on user-defined rules. My role as the main developer involved enhancing and completing a partially developed script, ultimately transforming it into a reusable Python library. This tool processed a corpus comprising 1 billion words, demonstrating its scalability. While initially developed for Slovenian, Cordex has been extended to support other languages, leveraging the Universal Dependency CONLLu format.

Key Details

  • Role: Main developer, responsible for enhancing, completing, and transforming the script into a reusable Python library.

  • Achievements: Enhanced user-defined rule options, implemented inflectional data lookup, integrated API queries, processed a 1 billion-word corpus, and extended language support to handle diverse formats in multiple languages.

  • Capabilities: Use of multicore processing and SQLite for efficient handling of large corpora. Incorporation of rule-based collocation extraction and versatile support for multiple data formats and languages.

Project Outcome

The project culminated in the creation of a robust Python library proficient in extracting collocations from corpora of varying sizes.
