Project Description

Overview

STARK is a Python-based command-line tool developed for extracting statistics from dependency trees in CONLL-U formatted corpora. It efficiently calculates various statistics, including subtree counts based on user-defined or automatically generated query trees, while leveraging multi-core processing for speed.

Key Features

  • Custom and Automatic Queries: Users can define specific query trees or let the tool generate queries automatically from certain parameters, making it accessible to researchers without query-building expertise.
  • Multi-Core Processing: STARK accelerates execution for large corpora.

Project Outcome

As the sole/main developer throughout the project, I honed my skills in efficient software development. STARK has been instrumental in generating scientific language articles and serves as the foundational framework for an upcoming web application, showcasing its potential in advancing linguistic research.
