    Portfolio

    Data Extraction / Natural Language Processing / Text Analysis / Tool Development

    CORDEX - a collocation extraction library

    The Cordex project led to the creation of a Python library tailored for extracting collocations based on user-defined rules. This versatile tool supports various text formats, processes datasets in multiple languages, and offers efficient data processing features, including large corpora handling and inflectional data lookup.

    Data Science / Machine Learning / Natural Language Processing / Text Analysis / Tool Development

    Classla

    Guided the development of a tailored Stanza library version for South Slavic languages, enhancing core functionalities such as tokenization, part-of-speech tagging, lemmatization, dependency parsing, and named entity recognition. Led the integration of rule-based techniques, retrained models, and introduced semantic role labeling models as the main developer, resulting in improved accuracy and enabling precise linguistic analysis.

    Data Science / Machine Learning / Natural Language Processing / Research

    Enhancing deep neural networks with morphological information

    Explored the enhancement of deep neural network models through the integration of morphological features for downstream tasks including named entity recognition, dependency parsing and comment filtering across multiple languages. Notable performance improvements were achieved in LSTM-based models, while BERT-based models showed limited enhancement.

    Data Science / Machine Learning / Natural Language Processing

    Standardness

    This project focused on identifying nonstandard language in Slovenian tweets using a unique standardness metric developed from a parallel corpus. By training SVMs and BERT-based neural networks, the project predicted tweet standardness, enabling the targeted selection of tweets for manual annotations.

    Natural Language Processing / Research

    Cross-Lingual Embedding Alignment Using Anchor Points

    This project explored cross-lingual embedding alignment using anchor points, enabling language model training and application across different languages. Findings highlighted the effectiveness of various anchor point sources, especially in aligning linguistically similar languages.

    Data Extraction / Natural Language Processing / Text Analysis / Tool Development

    STARK - Dependency Parsing Statistics Extractor

    A Python command-line tool for efficiently extracting statistics from dependency trees in CONLL-U formatted corpora. It equips researchers with custom and automatic query generation and multi-core processing capabilities, enhancing linguistic research.

    Data Extraction / Text Analysis / Tool Development

    LIST - Corpus analyzer

    Development of the LIST corpus extraction tool, a versatile program designed for extracting statistics from text corpora. It facilitates in-depth data analysis at various linguistic levels and is actively employed for comparing statistics across diverse corpora.

    Data Science / Machine Learning / Natural Language Processing / Research

    Stress assignment

    This project tackled the intricate task of stress assignment for Slovene words, utilizing machine learning and deep neural networks to achieve an accuracy of 87.62% for stress predictions. The model was used for annotations in the largest morphological lexicon for Slovenian, enhancing accessibility and accuracy in stress assignment for linguists and language enthusiasts.
